Linear regression is one of the most simple machine learning methods. The most common way a linear regression model is fit to data is by minimizing the residual sum of squares (RSS). This is also called the sum of squared error (SSE)...and sometimes the sum of squared residuals (SSR).

The residual sum of squares is a "cost function," which is calculated as follows:
RSS = √  ∑(yi - ŷ)2

where yi is the y value for the ith sample and ŷ is the linear regression model's prediction of the same sample. Therefore, the RSS's magnitude is a function of how difference each sample's predicted value is from the actual value. If the linear regression model perfectly overlaps with all samples, then the RSS would be 0.
