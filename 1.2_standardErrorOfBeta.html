Standard Error of a Single Beta Coefficient

When a linear regression model has been solved for the minimum cost function, one is left with the question, is the result significant? Futhermore, was the x predictor significantly correlated with the target variable?

To answer this question, we can perform a hypothesis test on the coefficient that was solved for in the linear regression model. The null hypothesis is that the x predictor is not correlated with the target variable at all. This would mean that the beta coefficient would equal 0. Therefore, the t-statistic calculation looks at follows:
t-value =     β - 0
SE(β)
where β is the calculated beta coefficient from the linear regression model and SE(β) is the standard error of that coefficient. The t-value is then checked in a standard t critical value table for the significance of the beta coefficient. As with standard statistical significance, if the confidence level is above 0.95 (p-value = 0.05), then the beta coefficient is considered signficant.
