Linear Regression Cost Function

As we learned in the previous lesson, the residual sum of squares is a good metric to see how close the linear regression fit is to the observed data. In this lesson, we're going to use that to find the optimal fit. The least squares RSS is what we call a "cost function." In order to find the best beta coefficients to fit for the data, we find the minimum of the cost function.
