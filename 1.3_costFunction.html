Linear Regression Cost Function

As we learned in the previous lesson, the residual sum of squares (RSS) is a good metric to see how close the linear regression fit is to the observed data. In this lesson, we're going to use that to find the optimal fit. This means we need to find the beta coefficients that can "fit" the measured data the best.

One way to do this is by minimizing the RSS. The least squares RSS is call a "cost function" in machine learning terms. In order to find the best beta coefficients to fit for the data in this case, we find the minimum of the cost function (lowest sum of squares).
